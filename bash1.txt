В данном файле представлены bash-команды для выполнения в терминале. Для полноценной проверки работоспособности рекомендуется выполнять их строго в указанной последовательности.

В каждом пункте дана краткая характеристика поставленной задачи, после двоеточия приведена сама команда. В пп. 19.1 и 19.2 — альтернативные опции для достижения одного и того же результата.

1. Открываем домашнюю директорию: cd

2. Определяем имя папки, в которой находимся: pwd

3. Создаем в этой папке каталог test1: mkdir test1

4. Переходим в папку test1: cd test1

5. Создаем файлы 1, 2 и 3 внутри папки test1: touch {1,2,3}.txt

6. Проверяем содержимое папки test1: ls

7. Переходим в домашнюю директорию: cd

8. Создаем в домашней директории папку test2: mkdir test2

9. Удаляем папку test2: rmdir test2

10. Удаляем файл 2 из папки test1: rm test1/2.txt

11. Создаем в домашней директории папку test3 и добавляем в нее два файла: mkdir test3 && touch test3/{1,2}.txt

12. Удаляем папку test3: rm -r test3

13. Создаем в домашней директории папку test4: mkdir test4

14. Перемещаем файлы 1 и 3 из папки test1 в папку test4: mv test1/{1,3}.txt test4

15. Добавляем в файл 1 три строки со словами line: echo line > test4/1.txt && echo line >> test4/1.txt && echo line >> test4/1.txt

16. Смотрим содержимое файла 1: cat test4/1.txt

17. Добавляем в файл 3 три строки со словами line: echo line > test4/3.txt && echo line >> test4/3.txt && echo line >> test4/3.txt

18. Смотрим содержимое двух файлов (1 и 3) сразу: cat {test4/1,test4/3}.txt

19.1. Используя один из редакторов, заменяем все строки в файле 1 (выбираем, что хотим заменить, и правим вручную): nano test4/1.txt

19.2. Пример команды без обращения к редактору: echo "QA is my life" > test4/1.txt

Все представленные выше команды проверены на работоспособность.
